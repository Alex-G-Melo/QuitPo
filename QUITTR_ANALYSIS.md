## Core Features

### 1. Panic Button
The flagship emergency feature for managing urges.

**How it works:**
- User presses when feeling tempted
- Phone vibrates and flashes
- Camera activates to display user's own face
- Shows stark motivational messages over the camera feed
- Creates a moment of self-confrontation to break the compulsive cycle
- Redirects to available support channels

**Psychology:** Uses self-confrontation technique - seeing yourself during a moment of weakness creates powerful behavioral interruption.

---

### 2. Melius AI Therapist
24/7 AI-powered support system.

**Features:**
- Available anytime (3 AM support when human help isn't available)
- Trained on NoFap wisdom and addiction recovery content
- Provides personalized guidance based on user emotions and triggers
- Private chat interface similar to ChatGPT but specialized for porn recovery
- Real-time support during cravings

**Limitations:**
- Limited messages per day (premium unlocks more)
- Some users report preferring human connection over AI

---

### 3. Streak Tracking & Progress Visualization

**Components:**
- **Streak Counter**: Tracks consecutive days without pornography
- **90-Day Target**: "Breakthrough point" milestone based on neuroscience
- **Progress Ring**: Fillable ring showing 30-day challenge completion
- **Life Tree Feature**:
  - Virtual plant that grows with your streak
  - Visual metaphor: grows from seed to tree
  - Relapse "kills" the plant - creating emotional investment
  - Progression stages: Seed → Enlightenment → Ascendant → Guardian → Fortress → Momentum → Pioneer → Nirvana

**Psychology:** Exploits the human motivation of maintaining unbroken chains and visible progress.

---

### 4. Advanced Content Blocker

**Capabilities:**
- Blocks 1M+ adult websites automatically
- Custom site blocking (add any URL)
- Keyword blocking (e.g., "gambling" blocks all pages containing the word)
- Real-time image filtering
- Safe search enforcement across all major search engines
- Works in Chrome incognito mode
- Integrates with iOS Screen Time
- Blocks apps and social media triggers

**Free Version Limits:**
- Block up to 3 websites
- Block up to 3 keywords

**Whitelist:** Allows unblocking accidentally flagged legitimate sites.
