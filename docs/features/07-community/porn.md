- Explicit website names

- Names of performers

- External links (especially to adult sites)

## Community Guidelines (Porn-Specific)

### Encouraged Content
- Sharing milestones and victories
- Discussing urge management techniques
- Asking for support during struggles
- Celebrating others' progress
- Discussing the science of recovery

### Not Allowed
- Describing specific pornographic content
- Mentioning website or performer names
- "Testing" if anyone wants to talk about porn
- Glorifying past use
- Seeking accountability partners for inappropriate purposes

### Gray Area (Moderator Discretion)
- Discussing PIED (porn-induced ED) - allowed if clinical
- Relationship intimacy challenges - allowed if recovery-focused
- Trigger descriptions - allowed if vague and educational
