```

• How do you want to feel when you're with people?
• What matters more than scrolling?

Example:
"I want to be present when I'm with my kids
instead of always being on my phone."
```

## Personalized Recommendations

Based on quiz answers:

| Answer | Recommendation |
|--------|----------------|
| > 5 hours usage | Aggressive friction mode |
| TikTok problem app | TikTok friction + Social Safety guide |
| Morning pickup | Morning phone-free challenge |
| Before sleep | Bedtime mode, phone outside bedroom |
| During work | Focus mode during work hours |
| Boredom trigger | Alternative activity suggestions |
| Comparison/anxiety | Grayscale mode, hide likes |
| > 100 pickups | Pickup awareness notifications |

## Quiz Data Usage

How quiz data personalizes the experience:

| Data | Used For |
|------|----------|
| Current usage | Baseline, progress tracking, goal suggestions |
| Problem apps | Friction mode setup, Social Safety priorities |
| Triggers | Notification timing, friction triggers |
| Reasons | Motivation messages, milestone celebrations |
| Pickup estimate | Pickup tracking baseline |
| Usage goal | Progress tracking, achievement unlocks |
| Personal why | Friction messages, daily reminders |
