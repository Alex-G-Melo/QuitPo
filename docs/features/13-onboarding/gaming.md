## Quiz Data Usage

How quiz data personalizes the experience:

| Data | Used For |
|------|----------|
| Daily hours | Baseline, time reclaimed calculations |
| Game types | Specific guidance (MMO vs single-player) |
| Triggers | Notification controls, friction setup |
| Reasons | Motivation messages, milestone tracking |
| Social importance | Guild departure guidance, friend messaging |
| Previous attempts | Approach recommendations, messaging tone |
| Personal why | Panic button, friction messages |
| Real-life goals | Achievement tracking, progress motivation |
