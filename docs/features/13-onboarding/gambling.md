## "Your Why" Prompts

Example prompts for gambling addiction:

```
Consider:
• What would financial peace of mind feel like?
• Who are you trying to protect?
• What would you do with the money you save?

Example:
"I want to pay off my debts and stop lying
to my family. They deserve honesty."
```

## Personalized Recommendations

Based on quiz answers:

| Answer | Recommendation |
|--------|----------------|
| Sports betting | Enable sports event alerts, block sports betting ads |
| Payday trigger | Payday check-in reminders |
| Watching sports | Social Safety for sports content |
| Financial stress trigger | Money saved tracker, financial resources |
| After losing | Chase-loss prevention messaging |
| Debt recovery reason | Financial milestone tracking |
| Social media ads | Social Safety platform guides |

## Quiz Data Usage

How quiz data personalizes the experience:

| Data | Used For |
|------|----------|
| Duration | Messaging tone, expectations |
| Gambling type | Platform-specific blocking, content filtering |
| Triggers | Check-in timing, sports alerts, payday reminders |
| Reasons | Panic button messages, milestone celebrations |
| Previous attempts | Compassionate messaging, feature recommendations |
| Financial impact | Money saved tracking, recovery milestones |
| Personal why | Panic button, urge interruption |
