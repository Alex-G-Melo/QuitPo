## Shame-Sensitive Messaging

All messaging avoids shame language:

**Avoid:**
- "Porn addiction"
- "Your problem"
- "What's wrong with you"
- "Dirty secret"

**Use Instead:**
- "Your journey to freedom"
- "What you're working on"
- "Building new patterns"
- "Recovery path"

| Frequency | Baseline comparison, progress framing |
| Triggers | Check-in prompts, notifications, Alex context |
| Reasons | Panic button messages, motivation reminders |
| Previous attempts | Compassionate messaging, feature recommendations |
| Longest streak | Goal setting, achievable milestones |
| Personal why | Panic button, urge interruption |
