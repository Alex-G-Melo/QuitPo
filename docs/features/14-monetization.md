# Monetization

## Title
Subscription Model with Free Tier and Premium Features

## Description
QuitPo uses a freemium model with a generous free tier that provides core recovery functionality, and a premium subscription that unlocks advanced features, unlimited access, and enhanced personalization. The pricing strategy balances accessibility (recovery tools should be available to all) with sustainability (the app needs revenue to continue development and support).

## Problem Statement

Monetizing recovery apps presents unique challenges:

1. **Ethical Concerns**: Putting recovery tools behind paywalls feels exploitative.

2. **User Demographics**: Many users struggling with addiction have financial constraints.

3. **Value Perception**: Digital recovery tools are hard to value compared to therapy.

4. **Competition**: Free alternatives and built-in device features exist.

5. **Conversion Barrier**: Users resistant to subscriptions, especially for sensitive apps.

6. **Churn Risk**: Recovery apps have natural churn as users "graduate."

## Solution

### 1. Generous Free Tier
Core recovery tools always free:
- Streak tracking
- Panic button
- Basic check-ins
- Limited AI chat (10/day)
- Basic breathing exercises
- Community read access

### 2. Premium Value
Clear value proposition:
- Unlimited AI therapist
- Full education library
- All meditations and sounds
- Advanced analytics
- Premium challenges
- Priority support

### 3. Accessible Pricing
- Regional pricing (PPP adjusted)
- Annual discount (40%)
- Student discount (40%)
- Hardship program

### 4. Ethical Approach
- No dark patterns
- Easy cancellation
- Grace period after lapse
- No guilt messaging

import Purchases from 'react-native-purchases';

// Initialize

const offerings = await Purchases.getOfferings();

## Ethical Guidelines

### What We DO
- Clear pricing, no hidden fees
- Easy cancellation (one tap to store)
- Grace period (7 days after lapse)
- Hardship program for those in need
- Free tier has real value
- No feature regression (grandfathered)

### What We DON'T Do
- No guilt messaging ("You'll lose your streak!")
- No artificial urgency ("Offer expires in 10 minutes!")
- No required payment info for trial
- No confusing plan names
- No hidden auto-renewals
- No paywalling crisis features (panic button always free)
